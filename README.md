##  1. Fechadura Eletrônica IoT

Sistema de controle de acesso desenvolvido para Raspberry Pi Pico W com interface web embarcada.

### Conceitos aplicados
- Sistemas embarcados
- Interface web
- Autenticação de usuários
- Hash de senha
- Registro de logs
- Armazenamento em memória flash

### Lógica do sistema

1. Usuário acessa o ponto de acesso
2. Usuário realiza login na interface da fechadura
3. Sistema recebe usuário e senha
4. A senha é convertida para hash
5. O hash é comparado com registros armazenados
6. O acesso é liberado ou negado
7. O evento é registrado em log
8. O usuário pode destrancar a fechadura com inserção de senha padrão pela interface web

Pasta do projeto: [Fechadura Web Embarcada](./fechadura_web_embarcada)

## 5. Projeto em Java  Sistema de Consultas Médicas – Arquitetura Hexagonal

Projeto desenvolvido em Java aplicando o padrão de Arquitetura Hexagonal (Ports and Adapters) para organização e desacoplamento das camadas do sistema.

O sistema simula o gerenciamento de consultas médicas, permitindo organizar informações de pacientes, médicos e agendamentos.

### Conceitos aplicados

- Programação orientada a objetos
- Arquitetura Hexagonal (Ports and Adapters)
- Separação de responsabilidades
- Organização modular do código
- Lógica de gerenciamento de consultas

Pasta do projeto: `projeto-java`
